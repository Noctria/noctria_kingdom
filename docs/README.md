# 👑 Noctria Kingdom

AIによる為替戦略王国「Noctria」のトレーディングシステムへようこそ。

本プロジェクトは、複数のAI戦略（臣下）と統合判断AI（王）が連携し、Airflowを通じて自律的なトレード戦略運用を実現する大規模AIトレーディングシステムです。
